# Frontend Web Application (Angular)

## Overview
Angular-based web application for the Car E-Commerce platform.

## Prerequisites
- Node.js 18+
- Angular CLI
- npm or yarn

## Setup

```bash
# Install dependencies
npm install

# Run development server
ng serve

# Build for production
ng build --configuration production
```

## Project Structure

```
src/
├── app/                # Application source
├── assets/            # Static assets
├── environments/      # Environment configurations
└── styles.scss        # Global styles
```

## Features
- Product catalog
- Shopping cart
- Order management
- User authentication
- Admin dashboard
- Real-time chat
- AI recommendations

## Development

### Running Tests
```bash
# Unit tests
npm test

# E2E tests
npm run e2e
```

### Code Style
```bash
# Lint
npm run lint

# Format
npm run format
```
